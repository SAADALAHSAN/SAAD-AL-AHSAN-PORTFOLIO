## 📝 Description
A high-performance, cinematic portfolio website built to showcase technical depth in Web3, Frontend Development, and Strategic Vision. This project leverages modern 3D graphics and smooth motion design to create an immersive user experience that bridges technical complexity with aesthetic excellence.

## 🔗 Live Demo
Check out the live site here: **[saad-portfolio-bice.vercel.app](https://saad-portfolio-bice.vercel.app)**

## ✨ Features
- **3D Interactive Elements**: Built with Three.js and React Three Fiber for immersive visual depth.
- **Cinematic Animations**: Smooth transitions and scroll-triggered animations powered by GSAP and Framer Motion.
- **Responsive Design**: Fully optimized for all devices using Tailwind CSS v4.
- **Ultra-Fast Performance**: Built on Vite for near-instant load times and high efficiency.

## 🛠 Tech Stack
| Category | Technology |
| :--- | :--- |
| **Core** | React 19, Vite |
| **Styling** | Tailwind CSS v4 |
| **Animations** | GSAP, Framer Motion |
| **3D Rendering** | Three.js, React Three Fiber |
| **Icons** | Lucide React, React Icons |

## 🚀 Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SAADALAHSAN/SAAD-AL-AHSAN-PORTFOLIO.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd saad-portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
